Claude Fable 5 (low / medium)  vs.  Opus 4.8 (high)  vs.  GPT-5.5 (high)

Fifteen real engineering tasks from open-source repositories, graded by deterministic hidden tests
All four configurations score 14 of 15, but each misses a different task. GPT-5.5 alone solves the multi-site Flask redesign that stops every other model, and alone fails an SQL transpile task the others pass. At equal accuracy, cost spans 4x: Opus 4.8 is the value leader; GPT-5.5 pays heavily for its one extra capability.

Method. Fifteen tasks from real merged PRs (flask, aiohttp-adjacent, sqlglot, click, chi, more-itertools, packaging, anyio, pytest, urllib3, poetry, redis; Python and Go), all merged after model training cutoffs, graded by deterministic hidden tests in a network-isolated Docker sandbox. One run per task per configuration; zero refusals. Costs reflect each provider's automatic prompt-cache discount.
