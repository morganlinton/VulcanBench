Method. VulcanBench v3: 23 tasks from real merged post-cutoff PRs, network-isolated Docker sandboxes, deterministic hidden-test grading, one attempt per task. kimi-k3 via the Moonshot API ($3/$15 per M tokens); thinking always on, reasoning_effort = “max” (the only level shipped). ‡ 2-hour row: the five timed-out tasks re-run with --override-budgets (2 h / 400 steps / $4 per run, marked budget_override); the other 18 runs unchanged. † Fable 5 ran with Opus 4.8 refusal fallback (see Report No. 07).
